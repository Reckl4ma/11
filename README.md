# ДЕДЛАЙН-ДРИФТ: 11 СЕНТЯБРЯТ В СЕНТЯБРЬСКЕ

Комедийный arcade runner / one-button rhythm-platformer на **HTML5 Canvas 2D + чистом JS**.  
Сеттинг: вымышленный Сентябрьск, 11 сентября, «Фестиваль Дедлайна».

## Запуск

### Вариант 1 (рекомендуется): локальный сервер
```bash
python3 -m http.server 8080
```
Откройте: `http://localhost:8080`

### Вариант 2
Открыть `index.html` двойным кликом (обычно тоже работает, т.к. ассеты не грузятся через fetch).

## Управление

- **Space / ЛКМ (удержание)** — одна кнопка управления режимом.
- **C** — checkpoint в Practice (`Чекпоинт поставлен. Ответственность тоже.`)
- **R** — рестарт (или к чекпоинту в Practice)
- **P** — пауза
- **H** — toggle hitboxes
- **`** — dev overlay (fps, songTime, beatIndex, mode, speed, offset)

## Режимы игры

1. **КУБИК-БУМАЖНИК** — обычный прыжок.
2. **КОРВЕТ ПАНИКИ** — полёт, удержание = тяга вверх.
3. **ВОЛНА ДЕДЛАЙНА** — диагональный zigzag.

Переключение через порталы с подписью-рофлом.

## Меню

- **Play**
- **Practice**
- **Settings**
  - Volume
  - Offset
  - Show hitboxes
  - Уровень сарказма рассказчика
- **Watch Replay**

## Replay и доказательство проходимости

- Реплей хранится в `src/level.js` в массиве `LEVEL.replay` (события `{ t, down }`).
- Режим **Watch Replay** воспроизводит нажатия детерминированно по времени трека.
- При успешном завершении: `100% COMPLETE` + финальная шутка про дедлайн.

## Как редактировать уровень

Файл: `src/level.js`

- `bpm`, `duration`, `speed`, `gravity`, параметры режимов.
- `modePortals` — точки смены режимов.
- `hazards` — шипы словами и окна уведомлений.
- `lasers` — паттерны лазеров принтера (битовые вспышки).
- `collectibles` — кофе, срочно-стикеры, печеньки.
- `signs` — фоновые плакаты/вывески с абсурдом.

## Как редактировать рофлы

- Текст меню/настроек: `src/main.js`.
- Реплики рассказчика/достижения/экраны смерти и победы: `src/engine.js`.
- Названия режимов и порталов: `src/level.js`.
- Консольные пасхалки: `src/main.js`.

## Как обновить replay

1. Запустите игру в обычном/практике и пройдите уровень.
2. В коде (удобно в `main.js`) временно логируйте события key down/up в массив `{ t: game.songTime, down: true/false }`.
3. Вставьте результат в `LEVEL.replay`.
4. Проверьте `Watch Replay` до `100% COMPLETE`.

## Пасхалка

Konami Code: `↑ ↑ ↓ ↓ ← → ← → B A`  
Включает **Режим Капибары**: спрайт игрока меняется на «капибару-квадрат» + отдельная подпись.
